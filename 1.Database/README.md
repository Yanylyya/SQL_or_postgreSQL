# Library Database Project

This project contains a relational database for managing information about books, authors, genres, and languages.

## Database Structure

The database contains the following tables:

- books
- authors
- genres
- languages
- book_authors
- book_genres
- book_languages

These tables are connected using many-to-many relationships.

## Features

The database allows you to:

- store information about books
- connect books with authors
- categorize books by genre
- store available languages

## Example Queries

Some example SQL queries included in this project:

- Show all books with their authors
- Find books in a specific genre
- Count how many books each author wrote
- Count books by genre
- Show full information about books (title, author, genre, language)

## Business Questions

This project answers several analytical questions, such as:

- Which authors wrote the most books?
- Which genres contain the most books?
- What are the newest books in the database?
- Which books belong to the Fantasy genre?
- What languages are available for the books?
